E038 Autogenerated Attribute Name Conflict
Errors and Warnings
noreferences

@@description

<h2>Umple semantic error issued when a name conflict occurs between an attribute and
an autogenerated attribute in the same class</h2>

<p>
Some attribute names are autogenerated with the declaration of an attribute. Using
an attribute name that has already been generated will issue the error.
</p>
