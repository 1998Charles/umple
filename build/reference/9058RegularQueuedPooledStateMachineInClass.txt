E058 Regular Queued and Pooled State Machines in Class
Errors and Warnings
noreferences

@@description

<h2>Umple semantic error reported when a class contains one or more queued state machine, pooled state machine and
regular state machine</h2>

<p>
An Umple class cannot contain at once a queued state machine, a pooled state machine and a regular state machine. < /br>
The state machines must be distributed in more than one class.
</p>
