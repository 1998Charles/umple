E096 Enumeration and Class Name Conflict
Errors and Warnings
noreferences

@@description

<h2>
Umple semantic error reported when an enumeration has the same name as the class it is used in.
</h2>

<p>
In Umple, enumerations must not have the same name as the class they are used in.
</p>
