Class Definition
Classes and Interfaces

@@description
<p>      
A class definotion defines an object-oriented  class available for use in your system.  A class can be 
composed of many things including: 
</p>

<ul>
<li>Specification of inheritance from a superclass using the isA keyword.</li>
<li>Dependencies</li>
<li>Attributes</li>
<li>Associations</li>
<li>Methods containing "code" in a native language (e.g. Java)</li>
</ul>

@@syntax
class [name] { [[classContent]]* }

@@example

class Student
{
  firstName;   // attribute - defaults to String
  lastName;
  Integer number;  // attribute with type Integer
  * -- * Address;  // Many-to-many association
  
  public String fullName()  // Method, whose content is not processed by Umple
  {
    return getFirstName()   " "   getLastName();
  } 
}

class Address
{
  String[] line;  // Multi-valued attribute
}//$?[End_of_model]$?

class Address
{
  position 47 160 109 64;
}

class Student
{
  position.association Address__Student 30,100 30,0;
  position 47 23 141 100;
}

@@endexample
