@@description
<p>      
A entity can be one of the following. They all are indicated by their name followed by a block of code enclosed in braces (curly brackets) { }
</p>

<ul>
<li>Class definitions (e.g. Student)</li>
<li>Association definitions</li>
<li>Association class definitions (e.g. a Booking that is both an association and a class at the same time)</li>
<li>Interface definitions</li>
</ul>
