Tracing

@@description
<p>Data collected while tracing ranges from a simple logging mechanism into a more sophisticated tracing mechanism that involves the record of lower level events triggered in user and kernel spaces. Umple provides tracing using different tracers. Tracers can be categorized into two main categories: <br>
<b>- Primitive: </b> <br>
  -- Console <br>
  -- File <br>
<b>- Advanced: </b> <br>
   -- LTTNG <br>
   -- Dtrace <br>
</p>

@@syntax

[[traceType]]

@@example

// this example injects traces using File tracer
tracer File;

class Student
{
  Integer id;
  trace id;
}

@@endexample
