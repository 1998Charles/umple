classDefinition
Entities

@@description
<p>      
The class command defines an object-oriented  class available for use in your system.  A class can be 
composed of many things including: 
</p>

<ul>
<li>Inheritance</li>
<li>Dependencies</li>
<li>Attributes</li>
<li>Associations</li>
<li>Arbitrary "code" in a native language (e.g. Java)</li>
</ul>

@@syntax
class [name] { [[classContent]]* }

@@example

class Student
{
  firstName;
  lastName;
  Integer number;
  * -- * Address;
  
  public String fullName()
  {
    return getFirstName() + " " + getLastName();
  } 
}

class Address
{
  String[] line;
}
@@endexample
