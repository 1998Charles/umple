E073 Duplicate Parallel State Machine Name in Nested State Machine
Errors and Warnings
noreferences

@@description

<h2>Umple semantic error reported when parallel state machines in the same nested state machine have the same name.</h2>

<p>
In Umple, parallel state machines within the same nested state machines must have different names.
</p>
