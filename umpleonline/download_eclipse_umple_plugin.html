Below are instructions for installing Umple, a model-oriented programming technology that lets you model in code. Here is more information about Umple, or try it hassle free with our UmpleOnline tool

Requirements

For Eclipse

  • Indigo Eclipse with Modelling tools. The installation instructions below will also require you to install Jet and XText.

For command-line use

  • An up-to-date JVM.

Directions

  • The latest Umple Downloads can be found here. This includes the Eclipse Plugin. It also includes a Jar for command-line compilation of Umple.

For Eclipse

  • Follow the directions on this page to install Umple in Eclipse.

For command-line use

  • Run 
    java -jar umple.jar <arguments>
  • Specify your umple file as an argument.
