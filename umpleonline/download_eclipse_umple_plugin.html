Below are instructions for installing Umple, a model-oriented programming language that lets you model in code. Here is more information about umple, or try it hassle free with our UmpleOnline version

Requirements

  • Java 5 or later (Mac OS X users: Eclipse 3.4 does not run on Java 6)
  • Eclipse 3.4 or 3.5

Install Eclipse

  • If you don't have Eclipse 3.5 (Galileo) or newer, install it from http://eclipse.org/downloads/
  • Start Eclipse

Add the Umple Update Site

  • Open the Install New Software dialog (Help > Install New Software...)
  • Add the Umple update site: http://cruise.site.uottawa.ca/umpleonline/UmpleUpdateSite/
  • Now that you have added the Umple update site, select the following features from the Umple update site:
    • UmpleEditorAndCompiler
  • Push the "Next..." button
  • Review the items to be installed and push the "Finish" button
  • Acknowledge that you trust the security certificate and push the "OK" button
  • Accept to restart Eclipse to make the changes effective.

Check The Installation

  • Launch Eclipse with a new project (any project)
  • Look for the Umple menu and the compile button
namespace example; class Student { Integer id; name; 0..1 -- 0..1 Mentor; } class Mentor { }
  • Write a small Umple application (you can use the example given above)
  • Click on Umple Menu > Compile Umple (or click on the Umple icon)
  • Your Java classes should be generated
  • Refer to the Umple Home Page to learn more about Umple
